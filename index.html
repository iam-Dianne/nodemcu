NodeMCU

NodeMCU is an open-source LUA based firmware developed for the ESP8266 wifi chip. By exploring functionality with the ESP8266 chip, NodeMCU firmware comes with the ESP8266 Development board/kit i.e. NodeMCU Development board.

F E A T U R E S

Wi-Fi

NodeMCU is powered by the ESP8266 or ESP32 chip, which has built-in Wi-Fi, making it perfect for IoT (Internet of Things) projects.

Lua and Arduino IDE Support

NodeMCU initially used Lua scripting language for programming, making it easy to write IoT applications. Later, it gained support for Arduino IDE.

GPIO (General Purpose Input/Output) Pins

The NodeMCU board has multiple GPIO pins (typically 10-16 pins depending on the version), allowing you to connect various sensors, actuators, LEDs, and other components.
