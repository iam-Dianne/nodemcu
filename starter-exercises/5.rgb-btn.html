RGB Lights with Button

Demonstrates the colors from RGB LED lights triggered by buttons.


Wiring Diagram






Code Snippet

  • The setup() function initializes and sets the initial values, while the loop() function loops the program, allowing it to change.
  • The pinMode() function configures the specified pin to behave either as an input or an output.
  • The digitalWrite() function gives value, either HIGH or LOW to a digital pin.
  • The digitalRead() function reads the value from a specified digital pin, wither HIGH or LOW.

RGB Lights with Button

                            

void setup() {
    pinMode(D0, OUTPUT);
    pinMode(D1, OUTPUT);
    pinMode(D2, OUTPUT);
    pinMode(D3, INPUT_PULLUP);
    pinMode(D4, INPUT_PULLUP);
    pinMode(D5, INPUT_PULLUP);
    }
    
    int buttonstate;
    
    void loop(){
    buttonstate = digitalRead(D3);
    if (buttonstate == LOW) {
        digitalWrite(D0, HIGH);
    }
    else {
        digitalWrite(D0, LOW);
    }
    buttonstate = digitalRead(D4);
    if (buttonstate == LOW) {
        digitalWrite(D1, HIGH);
    }
    else {
        digitalWrite(D1, LOW);
    }
    buttonstate = digitalRead(D5);
    if (buttonstate == LOW) {
        digitalWrite(D2, HIGH);
    }
    else {
        digitalWrite(D2, LOW);
    }
}
